Casting Line 1 Utility Costs By Item

Factory equipment can be modeled by state machines. Using a state model, resource cost-function can be modeled be time in a state. The basic state machine formalism is given as: OFF, PROCESSING(Busy), FAULTED(Down), BLOCKED and STARVED.

The Off state indicates that the process equipment is not in use (unpowered). The Processing state indicates the equipment is working to produce product. The Starved state indicates that there are missing input materials so the equipment is paused. If the storage facility for the process output is full, the process is in the Blocked state and the equipment delays processing. When equipment has a break down or fault, the process stops and the equipment is in the Faulted state. The numbers for repair for a piece of equipment when it has a break down or fault, indicate the equipment is immediately in the repair state although this is unlikely.
