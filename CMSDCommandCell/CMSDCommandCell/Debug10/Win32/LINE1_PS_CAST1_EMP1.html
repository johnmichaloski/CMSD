Planned times
0.00 POET Planned order execution time The planned order execution time shall be the planned time for executing an order.NOTE: It is often calculated from the planned run time per item multiplied by the order quantity plusthe planned setup time.
0.00 POT Planned operation time The planned operation time shall be the planned time in which a work unit can be used. The operationtime is a scheduled time.
0.00 PSUT Planned unit setup time The planned unit setup time shall be the planned time for the setup of a work unit for an order.
0.00 PBT Planned busy time The planned busy time shall be the operating time minus the planned downtime.NOTE: The planned down time may be used for planned maintenance work. The planned busy periodis available for the detailed planning of the work unit for expected production orders.
0.00 PRI Planned run time per unit The run time per item shall the planned time for producing one quantity unit.
Actual times
0.00 APWT Actual personnel work time The actual personnel work time shall be the time that a worker needs for the execution of a productionorder.
0.00 AUPT Actual unit processing time The actual unit processing time shall be the time needed for setup and for the production
0.00 AUBT Actual unit busy time The actual unit busy time shall be the actual time that a work unit is used for the execution of aproduction order.
720.00 AOET Actual order execution time The actual order execution time shall be the time difference between start time and end time of aproduction order. It includes the actual busy time, the actual transport and the actual queuing time.
0.00 APAT Actual personnel attendance time The actual personnel attendance time shall be the actual time that a worker is available to work onproduction orders. It does not include actual time for company authorized break periods (i.e. lunch). Itshall be the difference between login and logout excluding breaks.
0.00 APT Actual production time The actual production time shall be the actual time during which a work unit is producing. It includesonly the value-adding functions.
0.00 AQT Actual queuing time The actual queuing time shall be the actual time in which the material is either in transport orprogressing through a manufacturing process, i.e., the material is waiting for the process to begin.
574.00 ADOT Actual unit down time The actual unit down time shall be the actual time when the work unit is not executing order production although it is available.
574.00 ADET Actual unit delay time The actual unit delay time shall be the actual time associated with malfunction-caused interrupts, minor stoppages, and other unplanned time intervals that occur while tasks are being completed that lead to unwanted extension of the order processing time
0.00 ASUT Actual unit setup time The actual unit setup time shall be the time consumed for the preparation of an order at a work unit.
0.00 ATT Actual transport time The actual transport time shall be the actual time required for transport between work units.
0.00 AUPT Actual unit processing time The actual unit processing time shall be the actual production time plus the actual unit setup time.
0.00 AUBT Actual unit busy time The actual unit busy time shall be the actual unit processing time plus the actual unit delay time.
720.00 AOET Actual order execution time The actual order execution time shall be the time from the start of the order until the time of the completion of the order.
Maintenance elements
36.00 TBF Time between failures The time between failures shall be the actual unit busy time (AUBT) between two consecutive failures of a work unit including setup time and production time related to the orders being processed and without delay times.
156.00 TTR Time to repair Time to repair shall be the actual time during which a work unit is unavailable due to a failure.
36.00 TTF Time to failure The time to failure shall be the time between failures minus the time to repair.
11.00 FE Failure event count The Failure event count shall be the count over a specified time interval of the terminations of the ability for a work unit to perform a required operation.
156.00 CMT Corrective maintenance time The corrective maintenance time shall be the part of the maintenance time, during which corrective maintenance is performed on a work unit, including technical delays and logistic delays inherent in corrective maintenance (IEC 60050-191).
99.00 PMT Preventive maintenance time The preventive maintenance time shall be that part of the maintenance time during which preventive maintenance is performed on a work unit, including technical delays and logistic delays inherent in preventive maintenance (IEC 60050-191).
